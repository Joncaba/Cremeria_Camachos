
# Punto de Venta para Cremería

Esta es una aplicación desarrollada en Streamlit para gestionar un punto de venta en una cremería. Incluye módulos para ventas, productos, inventario, finanzas y turnos.

## Requisitos

- Python 3.8 o superior
- pip

## Instalación

1. Clona o descarga este repositorio.
2. Instala las dependencias:

```bash
pip install -r requirements.txt
```

3. Ejecuta la aplicación:

```bash
streamlit run main.py
```

## Módulos incluidos

- **Punto de Venta**: Registro de ventas con código de barras y selección de cliente.
- **Productos**: Gestión de productos, precios y stock.
- **Inventario**: Entradas y salidas de productos, alertas de stock bajo.
- **Finanzas**: Reportes de ingresos, egresos y márgenes de ganancia.
- **Turnos**: Asignación de turnos y visualización en display externo.

## Exportación de Reportes

- Reportes financieros e inventario disponibles en formato CSV.
